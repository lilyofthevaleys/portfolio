# Lanyard Assets Required

This directory needs two files for the Lanyard component to work:

## 1. `card.glb`
- A 3D model file in GLB format
- This is your ID card or badge model
- You can download placeholder assets from reactbits.dev or create your own
- Edit the card model using: https://modelviewer.dev/editor/
- Change the texture to customize your card

## 2. `lanyard.png`
- A PNG texture file for the lanyard band
- This image will wrap around the lanyard string
- Can be edited in any image editor (Photoshop, Figma, etc.)
- Recommended: Design with your school/company colors or branding

## Where to get these files:
- Download from the original reactbits.dev lanyard component repository
- Create your own card.glb using Blender or other 3D software
- Design your own lanyard.png texture

## Once you have the files:
1. Place `card.glb` in this directory
2. Place `lanyard.png` in this directory
3. The component will automatically load them
